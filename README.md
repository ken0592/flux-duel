# Flux Duel

Flux Duel is a small browser game prototype for a simultaneous-input area duel.

## Game

- 5x5 board
- 12 turns
- Blue and Red choose an action and direction at the same time
- Move claims 1 tile
- Dash claims up to 2 tiles
- Strike claims an adjacent tile and can stop an opponent that crosses it
- Gold is worth 3 points while owned
- The higher score after turn 12 wins

## Run

Open `index.html` in a browser, or serve the folder with any static file server.

For GitHub Pages, put these files at the repository root and enable Pages for the main branch root.
