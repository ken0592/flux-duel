# Flux Duel

Flux Duel is a small browser game prototype for a simultaneous-input area duel.

## Game

- 7x7 board
- 12 rounds
- Blue and Red lock actions at the same time
- The higher tile value after round 12 wins
- Random flux tiles add about 30 percent chaos without fully deciding the game

## Actions

- Step: move 1 tile with solid clash power
- Dash: move up to 2 tiles with weaker clash power
- Paint: claim your tile and a nearby tile
- Trap: arm a nearby trap for two rounds
- Guard: stay strong, block traps and static
- Blast: push or pin an adjacent opponent

## Flux Tiles

- Bonus: owned tile counts as 2
- Bloom: claims a plus shape
- Slip: moves a player one more tile
- Rift: teleports to an open tile
- Static: stuns unless guarded

## Run

Open `index.html` in a browser, or serve the folder with any static file server.

For GitHub Pages, put these files at the repository root and enable Pages for the main branch root.
